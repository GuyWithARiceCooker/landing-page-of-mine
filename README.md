# landing-page-of-mine

A simple, elegant landing page linking to my WordPress blog.

## 🌐 Live Site

Visit the landing page: [GitHub Pages](https://guywithericecooker.github.io/landing-page-of-mine/) (when GitHub Pages is enabled)

Or visit my blog directly: [My Blog](https://wordpress-233a5.wasmer.app/)

## 📝 What's Included

- **index.html** - Responsive landing page with links to blog and GitHub profile
- **Gradient Design** - Modern purple gradient background with clean white card layout
- **Mobile Friendly** - Fully responsive on all devices
- **Fast & Lightweight** - Pure HTML/CSS with no dependencies

## 🚀 Getting Started

### Option 1: Enable GitHub Pages
1. Go to Settings → Pages
2. Select `main` branch as source
3. Your site will be live at `https://guywithericecooker.github.io/landing-page-of-mine/`

### Option 2: Host Elsewhere
Simply download the `index.html` file and host it on any web server.

## 🎨 Customization

Edit `index.html` to customize:
- **Title and subtitle** - Update the welcome message
- **Blog link** - Change the WordPress URL
- **Colors** - Modify the gradient colors in the CSS
- **Text** - Update any content to match your brand
